One Blink, One Switch, One Voice

The 180-year-old code that modern disability technology still can't replace.

🕑 8 min read AccessibilityAACDisability TechALS

In 1995, a French journalist named Jean-Dominique Bauby suffered a massive stroke that left him with complete paralysis except for his left eyelid. To write his memoir — The Diving Bell and the Butterfly — his transcriptionist would recite letters of the alphabet in frequency order, and Bauby would blink when she reached the right one.

It took him 200,000 blinks. The book was published two days before he died.

Bauby's method was not Morse code — but it was the same idea: reduce communication to the simplest possible binary signal (open/closed, dot/dash) and build meaning from there. This is exactly what Morse code has been doing for people with severe physical disabilities since the 1970s.

Google's Morse Code Keyboard

In 2018, Google added Morse code as an official input method to Gboard, its keyboard app used by hundreds of millions of people. This came directly from a collaboration with Tania Finlayson, a woman who has had cerebral palsy from birth and has communicated using Morse code for over 30 years via a single cheek switch.

Finlayson can type faster in Morse code than most people type on a standard keyboard. She composes emails, texts, and social media posts — all via a device that detects a small muscle movement in her cheek and translates it into dots and dashes. The Google team built their keyboard after spending a week trying to match her speed.

"Morse code gave me a voice. It's not a historical curiosity — it's freedom." — Tania Finlayson, AAC user and Morse code advocate

How Switch-Based Morse Input Works

The two-switch setup:
  • Short press = dot
  • Long press = dash
  • Pause = letter break / word break

The switch can be activated by any controllable body part: thumb, cheek muscle, eyebrow, toe, tongue, or even breath pressure. This is why it works for people with extremely limited motor function.

Modern implementations are remarkably intelligent. Predictive text algorithms watch your Morse input in real time and start suggesting words after two or three letters — just like autocomplete on a smartphone. Combined with the muscle-memory speed that experienced users develop, the result is genuine conversational speed.

Eye-Tracking Systems and the Morse Principle

Modern eye-tracking communication devices — the kind Stephen Hawking used — operate on a related principle. The screen displays letters or symbols; the user dwells their gaze on a target to select it. At its core, this is dwell-time encoding: variation in the duration of a signal (gaze contact) carries meaning. Hawking's speech synthesiser, generating his famous synthesised voice, was ultimately powered by a binary signal from a cheek muscle — one switch, one contact at a time.

The conceptual line from Samuel Morse tapping a key in 1844 to Stephen Hawking discussing cosmology via a cheek twitch in 2018 is direct and unbroken.

ALS, Locked-In Syndrome, and the Last Line of Communication

For people in the late stages of ALS (motor neuron disease), communication often narrows to a single reliable muscle — an eyelid, a cheek, a toe. Morse code is frequently the method of choice because:

  • It requires only one controllable body part
  • It needs no visual display — it can be purely tactile or auditory
  • It works with extremely simple, robust hardware
  • It has no subscription fees, no proprietary lock-in, no cloud dependency
  • Skilled users can achieve 20–30 words per minute

Military Use: SERE and Prisoner Communication

Morse code has also functioned as an accessibility tool in a darker sense: for prisoners of war. During the Vietnam War, American POWs in the Hanoi Hilton prison used a tap code — a simplified grid-based variant of Morse — to communicate through walls, pipes, and concrete floors.

Commander Jeremiah Denton, shot down in 1965, was forced to appear in a televised propaganda interview. While speaking words scripted by his captors, he blinked his eyes in Morse code spelling T-O-R-T-U-R-E — the first confirmation the outside world received of how American POWs were being treated.

The tap code (simplified prison Morse):

A 5×5 grid of letters (C/K combined). One series of taps = row, pause, another series = column. So A = tap-tap (row 1), tap (column 1).

It can be sent through any surface that conducts vibration: walls, pipes, floors, even the sole of a shared shoe.
