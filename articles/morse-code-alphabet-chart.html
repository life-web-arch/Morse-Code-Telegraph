The International Standard

The Morse code alphabet used worldwide today is the International Morse Code defined by the International Telecommunication Union (ITU). It superseded the original "American" Morse code in the early 20th century and is used in all modern contexts including aviation, amateur radio, and emergency signalling.

Memory Tips

  • E (.) and T (-) are the simplest — one signal each. Learn these first.
  • S (...) and O (---) together spell SOS — the most important sequence to know.
  • Numbers always have exactly 5 signals. Numbers starting with dots (1–4) begin with that many dots, then fill with dashes.
  • The pattern for letters tends to mirror their frequency: common letters like E, T, A, N are shorter; rare letters like Q, X, Z are longer.
