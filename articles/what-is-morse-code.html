Definition

Morse code is a method of encoding text characters as sequences of two different signal durations — a short signal called a dot (dit) and a long signal called a dash (dah). These signals can be transmitted as sound tones, flashes of light, radio pulses, or even electrical pulses through a wire.

Every letter of the alphabet, every digit from 0–9, and many punctuation marks have a unique Morse code representation.

Who Invented Morse Code?

Morse code was developed in the 1830s and 1840s by Samuel F. B. Morse, an American painter-turned-inventor, and his colleague Alfred Vail. Their goal was to send messages over long distances using the newly discovered electrical telegraph wire.

The first official Morse code message was transmitted on May 24, 1844, famously reading: "What hath God wrought" — sent from Washington D.C. to Baltimore.

How Does It Work?

Morse code encodes meaning entirely through timing. The key rules are:

  • A dot is 1 unit of time.
  • A dash is 3 units of time.
  • The gap between symbols within a letter is 1 unit.
  • The gap between letters is 3 units.
  • The gap between words is 7 units.

So when you hear a classic "dit-dah-dit" (. - .) sequence, your brain maps it to the letter R. Skilled operators could copy Morse code at 30+ words per minute purely by ear.

Is Morse Code Still Used Today?

Yes, in several fields:

  • Aviation — VOR and NDB radio navigation beacons still transmit their station identifiers in Morse code, which pilots are trained to verify.
  • Amateur (Ham) Radio — Many countries still require or encourage Morse code proficiency for amateur radio licensing. CW (continuous wave) operation is an active part of the hobby.
  • Military & Emergency — The SOS distress signal (... --- ...) is universally understood. Morse can be transmitted with nothing more than a flashlight.
  • Accessibility — Assistive technologies allow people with limited mobility to communicate using two-switch Morse input.

Why Learn Morse Code?

  • It is a valuable survival skill — communicable with light, sound, or touch when no technology is available.
  • It is a fascinating piece of engineering and communication history.
  • It can be fun as a private code with friends or as a mental challenge.
  • It is required or recommended for amateur radio licences in many countries.
