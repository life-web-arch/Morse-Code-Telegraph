Morse Code Translator Guide

Everything you need to know about encoding, decoding, and using our free online Morse code translator.

What is a Morse Code Translator?

A Morse code translator is a tool that converts plain text into Morse code notation (dots and dashes) or converts Morse code back into readable text. Online translators like Morse Code Telegraph do this instantly as you type, with audio playback so you can hear the result.

Encoding: English to Morse

To encode text into Morse code:

  1. Each letter maps to a unique dot-dash sequence (see the Alphabet Chart).
  2. Letters within a word are separated by a single space in written notation.
  3. Words are separated by a forward slash with spaces:   /
Example — "HI BM":
.... .. / -... --

In our simulator, just type in the left text box and the Morse appears in the right box automatically — no button to press.

Decoding: Morse to English

To decode Morse code manually:

  1. Split the code by single spaces to get individual letter codes.
  2. Split by " / " (space-slash-space) to identify word boundaries.
  3. Look up each code in the reference chart.

In the simulator, you can type (or paste) Morse code directly into the right box and the English translation appears on the left instantly.

Using the Audio Decoder

Our tool can also decode Morse code from audio files:

  1. Click the "📁 Decode Audio" button.
  2. Select an MP3, WAV, WebM, or other audio file that contains Morse code tones.
  3. The tool analyses the audio waveform, detects the on/off pattern, and extracts the Morse code.
  4. The decoded English translation appears automatically.

Common Notation Rules

Written symbolMeaning
.Dot (short signal)
-Dash (long signal)
[space]Gap between letters
/ Gap between words

Tips for Accurate Decoding

  • Ensure spaces between letters are consistent — a missing space merges two letters into one unrecognised code.
  • For audio decoding, cleaner recordings (less background noise) produce more accurate results.
  • International Morse code uses only dots, dashes, and the specific gaps above. Avoid commas or other separators.
