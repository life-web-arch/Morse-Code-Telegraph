What is Morse Code Telegraph?

Morse Code Telegraph is a free, browser-based simulator and reference tool that lets anyone learn, translate, and practice Morse code — no downloads or accounts required. The tool runs entirely in your browser, works offline as a PWA, and supports encoding, decoding, audio playback, audio file decoding, and MP3/WAV export.

Our Mission

We believe communication history should be interactive and accessible. Morse code is not just a historical curiosity — it is still used in aviation (VOR identifiers), amateur radio, and accessibility devices. Our goal is to make it approachable for students, hobbyists, radio operators, and the simply curious.

What the Tool Does

  • Live Telegraph Key — Tap or click to transmit Morse code in real time, just like a real telegraph operator.
  • Text ↔ Morse Translator — Type English and get instant Morse code, or type Morse and get instant English.
  • Audio Playback — Hear your Morse code played back through an authentic sine-wave oscillator.
  • Audio Decoder — Upload an MP3, WAV, or other audio file and extract the Morse code from it automatically.
  • Export Audio — Download your Morse code as an MP3, WAV, or WebM audio file.
  • Reference Chart — Full ITU Morse code chart for letters, numbers, and punctuation with individual playback buttons.
  • Dark & Light Themes — Vintage sepia and dark charcoal themes to suit your environment.
  • PWA / Offline Support — Install on your phone or desktop and use it without internet.

Technical Details

The simulator is built with pure HTML, CSS, and vanilla JavaScript — zero frameworks, zero server-side processing. Audio is generated and processed entirely on the client using the Web Audio API. MP3 export uses the open-source lamejs library. The site is hosted on Vercel with a global CDN for fast loading everywhere.

Who Made This?

This tool was created as an educational side project with a focus on performance, simplicity, and authentic vintage aesthetics. It is maintained by BM and updated continuously based on user feedback.
